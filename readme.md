# CNN-Based Intelligent Security System  
## Differentiating Real Human Faces from Hyper-Realistic Disguise Masks

## 📌 Abstract
This project presents a Convolutional Neural Network (CNN) based intelligent security system designed to differentiate real human faces from hyper-realistic disguise masks. With the increasing use of realistic masks to bypass surveillance systems, traditional face recognition methods face major challenges. This system improves security by accurately classifying real faces and artificial masked faces.

---

## 🎯 Objectives
- To build a CNN model for face classification
- To distinguish real human faces from disguise masks
- To implement real-time detection using webcam
- To improve security in surveillance systems

---

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- TensorFlow
- Keras

---

## 🧠 Methodology
1. Collect dataset (Real faces and Mask faces)
2. Preprocess images (Resize, Normalize)
3. Train CNN model
4. Save trained model (.h5 file)
5. Perform real-time detection using webcam
6. Display prediction result on screen
